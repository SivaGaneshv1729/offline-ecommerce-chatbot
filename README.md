# Offline Customer Support Chatbot

A demonstration of using a locally-hosted LLM (Llama 3.2 3B) via Ollama to automate customer support while ensuring data privacy.

## Features
- **Privacy-First**: No data leaves your machine.
- **Prompt Engineering**: Comparison of Zero-Shot and One-Shot techniques.
- **Evaluation**: 20 e-commerce queries adapted from the Ubuntu Dialogue Corpus.

## Project Structure
- `chatbot.py`: Main script for querying Ollama.
- `prompts/`: Directory containing prompt templates.
- `eval/`: Directory with evaluation results and scoring.
- `report.md`: Experiment analysis and findings.
- `setup.md`: Installation instructions.
